# 네이버 호환 브랜드/컬러 규칙

## 핵심 컬러
| 토큰 | 값 | 용도 |
|------|----|----|
| naver/green | #03C75A | 네이버 연동 액션(지도 길찾기·네이버페이 영역 강조). 단, 페이 버튼 자체는 공식 에셋 사용 |
| naver/green-pressed | #02B350 | press 상태 |
| brand/primary (SpaceOS) | #0EA5B7 | SpaceOS 고유 1차 액션(분석·AI). 그린과 보색으로 충돌 회피 |
| ink | #1C2533 | 본문 텍스트 |
| muted | #6B7280 | 보조 텍스트 |
| surface | #FFFFFF / line #E3E9F2 | 카드·구분선 |

## 원칙
1. 네이버 그린은 **네이버와 연결되는 맥락**(지도·결제)에만 쓴다. 남발하면 네이버 사칭처럼 보인다.
2. SpaceOS 고유 기능(AI 추천·히트맵)은 brand/primary(teal)로 구분한다.
3. 네이버페이 버튼은 공식 에셋 그대로. 주변 여백·정렬만 우리 토큰(spacing/radius)으로 맞춘다.
4. 한글 가독성 최우선 → Pretendard, 본문 14–16px, 줄간격 1.5.
5. 대비 WCAG AA(4.5:1) 이상 — 그린 위 텍스트는 흰색.
